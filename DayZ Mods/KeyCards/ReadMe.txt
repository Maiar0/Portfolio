This is a mod that allows the server owner to configure places around the map to place Doors which take a specific item to open. Upon opening these doors place a chest with items inside of it for the person that opened the door. All aspects of this mod are configurable by the server owner, placement of all doors what a door item is, what a key is, what item is placed for the chest what items are placed inside. This mod also has options to Sounds alarm when door opens and this is also configurable to be on off how long to sound. The door can be configured to re close or not. This mod is very versatile in every aspect.
